Game Rules

You have 6 attempts to guess the 5 letter word

Enter words that exist using the keyboard

Note that a word can contain two or more identical letters

Pay attention to the colors:

Green - this letter is in the word and in a right place

Yellow - this letter is in the word but in a different place

Gray - this letter is not in the word
